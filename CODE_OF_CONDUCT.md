# Code of Conduct

## Our Standards

This project is a professional, technical environment. We expect all participants to:

- Be respectful and constructive in discussions, code reviews, and issue threads.
- Focus feedback on code and ideas, not on individuals.
- Accept that disagreements happen and that decisions are made in the interest of the project.
- Assume good intent until evidence suggests otherwise.

## Unacceptable Behavior

- Personal attacks, insults, or derogatory comments directed at contributors.
- Deliberate intimidation or harassment in any project space.
- Publishing private information about others without consent.
- Sustained disruption of discussions or reviews.

## Scope

This code of conduct applies within all project spaces: GitHub issues, pull requests, discussions, and any other communication channel used for project coordination.

## Enforcement

Violations may be reported by opening a GitHub issue marked `[conduct]` or by contacting the maintainer directly via GitHub. Reports will be reviewed and addressed within 7 days. Responses may range from a private warning to permanent removal from the project.
